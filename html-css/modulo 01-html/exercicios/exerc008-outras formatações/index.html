Principais Formatações

Negrito/Destaque

Negrito: usando negrito com a tag <b>,está tag é não semantica, contraindicada.

Destaque: usando destaque, dessa vez usamos a tag <strong>, que possui semantica.

Itálico/Ênfase

Itálico: exemplo de texto de itálico, usando a tag <i>, não semântica.

Ênfase: exemplo de texto com Ênfase, usando a tag <em>
